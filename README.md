
# Portfolio 

This repository contains the source code for my personal portfolio website.  
The portfolio showcases my projects, skills, achievements, and interests in technology, product, and consulting.

The website is designed to be clean, responsive, and easy to navigate, with a focus on clarity and storytelling rather than heavy visuals.

## Live Demo
(Will be updated once deployed on Vercel)


## Tech Stack
- **Framework:** Next.js / React  
- **Styling:** Tailwind CSS  
- **Version Control:** Git & GitHub  
- **Deployment:** Vercel  
---
## Features
- Responsive design for all screen sizes  
- Clear section-wise layout (About, Projects, Achievements, Contact)  
- Easy to update and scale  
- Optimized for performance and readability  
---
## Purpose of This Portfolio
This portfolio serves as:
* A personal space to document my learning journey
* A showcase of projects and problem-solving approach
* A professional introduction for internships, roles, and collaborations
